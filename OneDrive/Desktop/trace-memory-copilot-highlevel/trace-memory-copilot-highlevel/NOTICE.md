TraceMemory Copilot is an independently maintained project that demonstrates how to add long-term conversational memory to an AI assistant using Zep.

This repository started from a learning scaffold based on an open-source example in the **AI Engineering Hub** repository by patchy631 (MIT License). The codebase has been rebranded, restructured, and extended with additional features and UX improvements.
